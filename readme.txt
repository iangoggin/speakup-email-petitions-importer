== Installation ==

Use the automatic installer. Or...

1. Download and unzip the the plugin zip file.
2. Upload the `speakup-email-petitions-importer` folder to your `/wp-content/plugins/` directory
3. Activate SpeakUp! Email Petitions Importer through the "Plugins" menu in the WordPress admin.

== Frequently Asked Questions ==

= Where can I find a csv template ? =
Go to signatures > export to csv
The plugins imports files in the same format that the export
